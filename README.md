# draw-api

This project is a simple API for drawing shapes on a canvas. It uses Node.js and Express to create a server that listens for HTTP requests. The API allows you to draw circles, rectangles, and lines on a canvas by sending POST requests with the appropriate parameters.

## Installation

To install the project, follow these steps:

1. Clone the repository:

```
   git clone https://github.com/sebasballen31-ai/draw-api.git
```

2. Navigate to the project directory:

```
   cd draw-api
```

3. Install the dependencies:

```
   npm install
```

## Usage

To start the server, run the following command:

```
   npm start
```

The server will start and listen on port 3200.
